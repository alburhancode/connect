API endpoints for grading

Get all candidates that have payments verified

REQUEST
https://alburhanconnect.com/api/v1/list/verified/{batch}/{authority}
where {batch} can be
  • dec2019isb
  • dec2019isbw
  • dec2019lhr
  • dec2019khi
and authority can be
  • panel
  • ceo
EXAMPLE panel
https://alburhanconnect.com/api/v1/list/verified/dec2019isb/panel

EXAMPLE ceo
https://alburhanconnect.com/api/v1/list/verified/dec2019khi/ceo

Save Ceo / Panel Grading for one or more candidate

REQUEST
https://alburhanconnect.com/api/v1/grading/save/{authority}/{apikey}
where {authority} can be
  • ceo
  • panel
EXAMPLE usage for CEO
https://alburhanconnect.com/api/v1/grading/save/ceo/{apikey}

EXAMPLE usage for Panel
https://alburhanconnect.com/api/v1/grading/save/panel/{apikey}

REQUEST BODY
{ "email":"abc@gmail.com", "createdAt":"2019-11-15T12:52:10.413Z", "grading": [ { "grade":"3", "comments":"well done but need more" } ] }

where Grades (int) range in [ 1 | 2 | 3 | 4 | 5 | 6 | 7 | 0 ]
  • [1] for C
  • [2] for B
  • [3] for B+
  • [4] for A-
  • [5] for A
  • [6] for A+
  • [7] for Listener
  • [0] for none or not graded
